# Sourdough-Bread
Are you like me and have an old console like a Wii, PSP or Xbox 360 laying around (or being used) that you wish can run Fortnite, Hogwarts Legacy, Halo Infinite, or pretty much any PC game? Well, I wanted to run that on my 360. I know what you're thinking; "BuT wHy CaN't YoU jUsT pLaY iT oN yOuR PC dIrEcTlY?" To be honest, it will DEFINITELY run better that way. But, if you just want to say "I GOT FORTNITE ON A 20 YEAR OLD CONSOLE I FOUND IN MY BASEMENT," then this is for you. It's designed to be a remote desktop (RD) that can run in any old browser like Internet Explorer. NO MODDING IS REQUIRED! So now you can run PC games on a browser that might be older than you.

## Instructions!

Download the source code (boot.png and server.py). Install python from https://www.python.org/downloads/, and move it to PATH. You can find a tutorial of that on YouTube, it was pretty easy once I found out how to do it. Now, you need to run this command:
```bash
pip install flask mss pillow
```
Double click server.py and set your config.
It will show you IP address, and go to this address on your weak browser (case sensitive):
http://your.ip.is.here/

Now, you can run Fortnite on a browser that's old enough to be in a museum.
